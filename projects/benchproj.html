Bench Project

This is my bench project where I created a bench which could rotate which allows easy seating when it is raining, dirty, etc. This was made on AutoDesk Inventor and was my hardest project to date.

First, I worked on making the parts for the bench.

I created the seat by using a rectangular prism and then using the 3D fillet to make sure the bench was easy to be sat on.

For the legs, I made a rectangle and then used the filet command to make the rounded corners.

Then, I made a hole in the middle to fit the crank.
